# Automatic Water Level Controller

## Project Overview
This project automatically monitors and controls the water level in a tank using Arduino Uno and HC-SR04 Ultrasonic Sensor.

## Features
- Automatic Pump Control
- Manual Pump Control
- LCD Status Display
- Water Level Monitoring

## Components Used
- Arduino Uno
- HC-SR04
- LCD 16x2
- Relay Module

## Working Principle
1. Sensor measures water level.
2. Arduino calculates percentage.
3. Pump turns ON/OFF automatically.
