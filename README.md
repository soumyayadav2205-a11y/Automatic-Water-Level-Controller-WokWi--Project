# Automatic Water Level Controller

## Overview
This project automatically controls a water pump based on tank water level.

## Components Used
- Arduino Uno
- HC-SR04
- LCD 16x2
- Relay Module
- LED
- Push Buttons

## Features
- Automatic Pump Control
- Manual Mode
- LCD Monitoring
- Overflow Prevention

## Simulation Platform
Wokwi
